Getting started

Build the workspace, run the tests, and create your first solid.

Prerequisites

A recent stable Rust toolchain via rustup. No C/C++ toolchain, no system libraries — the kernel is pure Rust and compiles natively and to WebAssembly.

Build & test

# clone, then from the workspace root:
cargo build --workspace        # build every crate
cargo test  --workspace        # unit + integration suite
cargo clippy --workspace --all-targets   # lint (a CI gate; kept clean)

Handy aliases live in .cargo/config.toml:

cargo t     # == cargo test --workspace
cargo ck    # == cargo clippy --workspace --all-targets

Add it to your project

Depend on the openrcad facade crate — it re-exports every kernel layer as a submodule (foundation, geom2d, geom, topo, sketch, primitives, algo, document, mesh, exchange) plus a prelude. The interactive viewer (openrcad-render) is a separate dependency.

# Cargo.toml
[dependencies]
openrcad = { version = "0.1" }

Prefer thinner builds? Depend on the individual crates (e.g. openrcad-foundation, openrcad-topo) directly. See the crates map.

Your first solid

use openrcad::foundation::{Pnt, Dir, Ax1, Trsf};
use openrcad::primitives::make_box;

// A 10×20×30 box sitting on the origin.
let solid = make_box(&Pnt::new(0.0, 0.0, 0.0), 10.0, 20.0, 30.0);
assert_eq!(solid.shell().faces().len(), 6);
assert_eq!(solid.vertex_count(), 8);

// Rotate 90° about Z and read the moved AABB.
let rot = Trsf::rotation(&Ax1::new(Pnt::origin(), Dir::dz()), std::f64::consts::FRAC_PI_2);
let moved = solid.transformed(&rot);
let (min, max) = moved.bounding_box().corners().unwrap();

A complete pipeline

Primitive → blend → mesh → file, the way an application uses it:

use openrcad::foundation::{Pnt, Dir, Ax2};
use openrcad::primitives::make_cylinder;
use openrcad::algo::chamfer;
use openrcad::mesh::tessellate;
use openrcad::exchange::write_step;

let cyl  = make_cylinder(&Ax2::new(Pnt::origin(), Dir::dz()), 5.0, 12.0);
let part = chamfer(&cyl, 0.8)?;            // 45° frustum chamfers on both rims
let mesh = tessellate(&part, 0.05, 0.5);  // chordal + angular tolerance
write_step(&part, "part.step")?;
println!("{} triangles", mesh.triangle_count());

Blend builders (fillet, chamfer, shell_solid) return Result<Solid, BlendError>. Handle BlendError::UnsupportedShape (not a box/cylinder primitive) and BlendError::ParameterTooLarge { max, .. }. The full pattern is in the integration guide.

See it in a window

The openrcad-render crate (a separate dependency, not part of the facade) opens an interactive wgpu viewer — orbit / pan / zoom, 4× MSAA, an edge wireframe, and click-to-select:

// Cargo.toml: openrcad-render = { version = "0.1" }
openrcad_render::run_solid(&part, 0.02);  // (solid, chord error) — blocks until closed

Verify a result

Every modeling result should be a closed, two-manifold solid. Check it directly:

assert!(part.is_watertight());      // every edge shared by exactly two faces
assert!(part.validate().is_ok()); // every boundary loop closed + contiguous
