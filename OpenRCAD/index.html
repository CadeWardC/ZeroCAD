OpenRCAD

A lightweight, dependency-light geometric modeling kernel in pure, #![forbid(unsafe_code)] Rust — a refactor of OpenCASCADE's architecture, informed by truck.

pure Rust no unsafe serde-serializable native + WASM clippy-clean

What it gives you

OpenRCAD provides the foundation a CAD application needs — points, vectors, curves, surfaces, boundary-representation (B-Rep) topology, primitive builders, booleans, blends, tessellation, and STEP/STL exchange — without depending on any C++ kernel. Everything is owned, Clone, and Serialize, so a whole model round-trips to a single document blob.

Build geometry

Boxes and cylinders out of the box; lines, conics, and full NURBS curves/surfaces underneath, evaluated with de Boor and Boehm.

Combine it

Boolean union / intersection / difference over a BVH-accelerated intersection engine with robust, tolerant classification.

Blend & hollow

Fillet, chamfer, and shell a box or cylinder at any orientation — rolling-ball tori, frustum chamfers, offset-surface cavities.

Mesh & exchange

Adaptive parallel tessellation to triangle meshes; read/write STEP (AP242) and STL for interop with other tools.

View & pick

An interactive wgpu viewer: orbit / pan / zoom, 4× MSAA, a topological-edge wireframe, and click-to-select faces.

Sketch & document

2D profiles and a parametric document history (the .zcad format) — a Fusion/FreeCAD-style modeling spine on the kernel.

30-second taste

use openrcad::foundation::{Pnt, Dir, Ax2};
use openrcad::primitives::make_cylinder;
use openrcad::algo::fillet;

let cyl = make_cylinder(&Ax2::new(Pnt::origin(), Dir::dz()), 2.0, 6.0);
let rounded = fillet(&cyl, 0.5)?;  // rims become torus fillets
assert_eq!(rounded.face_count(), 11);

Blend builders return Result<Solid, BlendError>; see the integration guide for the full pattern.

Why another kernel?

truck proved a pure-Rust kernel is possible, but its boolean solver panics on some configurations, it has no fillet/chamfer builder, and smooth faces break its boolean path. OpenRCAD adopts OpenCASCADE's battle-tested module architecture and re-implements each layer in safe Rust — a kernel whose entire stack you own and can fix. It contains no OCCT or truck source; both are design references only.

Status: pre-alpha. The foundation, geometry, topology, primitives, intersection engine + BVH, booleans, tessellation, sewing, and STEP/STL layers are real and tested — plus an interactive viewer and a parametric document layer. Booleans are watertight on a wide range of inputs (with a documented partial-imprint frontier); fillet / chamfer / shell handle a box or cylinder at any orientation. See Status for the full picture.
